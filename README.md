## 📖 Overview

Conditionals let a program branch and make decisions based on whether an expression evaluates to `true` or `false`. This Swift playground is a short, hands-on learning example that walks through the core ways Swift expresses conditional logic — from a simple `if` check to compound conditions, the ternary operator, and `switch` statements with range matching. Open it, hit run, and read the console output alongside each snippet.

## 📌 What it covers

- **`if` statements** — running code only when a condition is met (e.g. `if firstNumber + secondNumber == 35`).
- **Compound conditions** — chaining multiple checks with a comma so all must hold (`if constant1 > 10, constant2 > 18`).
- **Ternary operator** — a one-line `condition ? valueIfTrue : valueIfFalse` for concise branching.
- **`switch` statements** — matching a value against multiple cases, including range patterns (`case 1..<3:`) and a `default` fallback.

## 🚀 Getting Started

```bash
git clone https://github.com/ahmetbostanciklioglu/Conditions.git
cd Conditions
```

Unzip `Conditions.playground.zip`, open `Conditions.playground` in Xcode, and press **⌘R** to run the code and view the results in the console.

- macOS with **Xcode** installed
- **Swift 5** (iOS playground target)

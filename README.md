# Conditions

**Condition:**

```
let firstNumber = 20
let secondNumber = 15
if firstNumber + secondNumber == 35 {
        print("condition is true.")
}
```

**Multiple Conditions:**

```
let constant1 = 14
let constant2 = 27
if constant1 > 10,  constant2 > 18 {
    print("Both are adult")
}
```

** Ternary operator:**
```
let ternary1 = 15
let ternary2 = 18
print(ternary1 == ternary2 ? "Ternary result is true" : "Ternary result is false")
```

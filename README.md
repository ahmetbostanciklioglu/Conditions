# Conditions

**Condition:**
```
let firstNumber = 20
let secondNumber = 15
if firstNumber + secondNumber == 35 {
        print("condition is true.")
}
```

**Multiple Conditions:**
```
let constant1 = 14
let constant2 = 27
if constant1 > 10,  constant2 > 18 {
    print("Both are adult")
}
```

**Ternary operator:**
```
let ternary1 = 15
let ternary2 = 18
print(ternary1 == ternary2 ? "Ternary result is true" : "Ternary result is false")
```

**Switch:**
```
let conditionCheck = 41
switch conditionCheck {
case -1..<1:
    print("0")
case 1..<3:
    print("2")
case 3..<5:
    print("4")
default:
    print("default")
}
```
